# 📱 Kiubo! – La evolución 2026 de "Yo"

**Kiubo!** es una aplicación web moderna que te permite guardar contactos y enviarles un mensaje SMS instantáneo con la palabra **"Kiubo!"**.  
Es la versión actualizada de la famosa app "Yo", pero con un toque fresco y la capacidad de enviar SMS reales mediante Twilio o usando el esquema nativo del teléfono.

## 🚀 Características

- ✨ Interfaz moderna con efecto glassmorphism y diseño responsive.
- 💾 Almacenamiento local de contactos (no se pierden al recargar).
- 📱 Envío de SMS de dos formas:
  - **Backend con Twilio**: envía SMS automáticamente a cualquier número del mundo (requiere configuración).
  - **Fallback nativo**: abre la app de SMS del dispositivo con el mensaje prellenado.
- 🛡️ Validación de números telefónicos y prevención de XSS.
- 🌐 PWA lista para instalar en dispositivos móviles (archivo `manifest.json` incluido).
- 🗺️ Sitemap XML para SEO.
